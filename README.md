# Politique de confidentialité - Fresh Coaching

Fresh Coaching collecte uniquement les données nécessaires au bon fonctionnement du service de coaching sportif et nutritionnel.

**Données collectées** : nom, prénom, numéro de téléphone, poids, objectifs sportifs.

**Utilisation** : ces données sont utilisées uniquement pour personnaliser les programmes de coaching et communiquer avec les clients via WhatsApp et Instagram.

**Conservation** : les données sont conservées pendant la durée de la relation commerciale.
